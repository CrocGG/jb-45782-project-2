Token List

Welcome to the Second Project of our course! This is a project made to depict one hundred crypto currencies utilizing an application programming interface (API) implementation from a database embedded in a site called CoinCap API Manager. The command-in-code invokes the tokens, and I added certain applications concerning employing the interactivity and following management of the tokens being chosen by the user.

  • The Dashboard contains a search block to search and filter certain tokens by the user volition. Pay attention that every letter inserted reduces the search pool interactively. Also, the token boxes themselves depict valuable information concerning them: their name and symbol from which you can filter desired tokens, 'More Info' rubric (featured exclusively on desktop applications) on which USD conversion currency is shown when being hovered and is updated by an every single minute renewing cache, and a blue switch to control the status of following a specific token.
  • The Coin Graphs tab illustrates a static graph containing only the tokens you have followed simultaneously - five max - depicting their prices graphically for the user to compare and contrast. When turning the switch off the corresponding graph will vanish. Press the 'unfollow' button in the dashboard tab to get a brand-new graph system afresh.
